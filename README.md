# Simon Game

A simple memory game built with **HTML**, **CSS**, and **JavaScript**.

The game generates a growing color sequence, and you need to repeat it correctly by clicking the colored buttons in the same order.

## Live Demo

Play here: https://saumyakhobragade.github.io/simon-game/

## Features

- Start game with any keyboard key
- Random color sequence generated each level
- Visual feedback for game sequence and user clicks
- Game over state with final score
- Tracks previous game scores
- Displays highest score from current session

## How To Play

1. Press any key to start the game.
2. Watch the flashed color carefully.
3. Click the color button that matches the sequence.
4. With each new level, one more color is added to the sequence.
5. If you click the wrong color, the game ends and your score is saved.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (DOM manipulation, events, arrays)
